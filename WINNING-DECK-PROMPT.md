# Победная дека Insider Guard — промпт для Claude Design
*Единый промпт. Данные ВЫВЕРЕНЫ по первоисточникам (`SOURCES.md`), структура заточена под 5 критериев AFM AI Hackathon (по 20 баллов — дека вручает судье доказательство по каждому). 10 слайдов, дека ОБРАМЛЯЕТ живое демо.*

> Как использовать: вставь блок ниже целиком в Claude Design. Если выдаст слоп — генери слайд-за-слайдом (сначала «визуальная система + слайд 1», утверди стиль, потом «тем же стилем — слайд N»). Числа на слайдах = числа на вкладке «Валидация» = слова спикера.

> Ты — арт-директор и дизайнер презентаций уровня топовых кибербез-вендоров (Wiz, Microsoft Sentinel, CrowdStrike). Создай **выигрышную** презентацию из 10 слайдов (16:9, тёмная тема) для защиты MVP на хакатоне.
>
> **ПРОГРАММА (под что побеждаем).** AFM AI Hackathon 2026, 24–25 июня, Алматы, КазНУ им. аль-Фараби. Организатор — **Агентство по финансовому мониторингу РК (финансовая разведка)**. Трек — **AI Shield** (защита данных и выявление киберугроз в реальном времени). Формат: защита 3 минуты + Q&A 2 минуты. Жюри — финразведка, кибербез-эксперты, возможен гос-юрист. Оценка — **5 критериев по 20 баллов = 100**:

> 2. наличие AI-модели, **независимой от иных сервисов**;
> 3. информационная безопасность + этические нормы;
> 4. практическая реализуемость + масштабируемость;
> 5. качество презентации и технической защиты MVP.
> Дека должна делать доказательство по КАЖДОМУ критерию очевидным — судья ставит балл по каждому, не заставляй его искать.
